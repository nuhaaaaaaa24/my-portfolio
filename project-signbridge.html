Real-time sign language translation bridging communication between the hearing and deaf communities through gesture-based AI.

Group Project 2025 Final Year
About the Project

SignBridge is a real-time sign language translation system developed as a final-year group project. The system uses a CNN-based gesture recognition model to detect and interpret hand signs captured through a live camera feed.

The goal was to create an accessible communication bridge — converting hand gestures into readable text in real time, reducing the barrier between the hearing and deaf communities. The project focused on high-accuracy gesture detection under varying lighting conditions and hand positions.

The frontend was built with HTML/CSS to display live video alongside the translated output, while Python powered the model inference pipeline with a Flask backend handling real-time data streaming.
