About the Project

BookBliss is a full-stack web application developed as a second-year group project, designed to help users connect, share reading experiences, and discover others with similar literary interests.

The platform combines social interaction with personal reading tracking, allowing users to manage reading lists and take part in community challenges through a user-friendly interface.

The frontend was built with Laravel Blade templates, while the backend uses MySQL to manage user data, books, and interactions. The project also emphasized teamwork and version control using Git.
