Individual Project 2023 Certification Project
About the Project

The Membership Management System is a backend web application designed to efficiently manage member records using a structured and scalable architecture. It supports full CRUD operations, allowing administrators to add, view, update, and delete member data.

Built with Spring Boot, the application follows a layered architecture that separates the controller, service, and data access layers. It integrates with a MySQL database for persistent storage and demonstrates key backend concepts such as RESTful API design and database integration.

While focused on core functionality, the system provides a foundation for future features like authentication, role-based access, and reporting.
